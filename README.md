Agenda de Contatos Simples (C)

(PROJETO AINDA EM ANDAMENTO E EM DESENVOLVIMENTO)

Este projeto consiste em uma Agenda de Contatos desenvolvida na linguagem C com o objetivo de praticar conceitos fundamentais como struct, vetor de structs, manipulação de strings, uso de ponteiros, modularização com funções e busca sequencial. O sistema atualmente permite adicionar contatos (preenchendo nome, telefone e email), listar todos os contatos cadastrados e buscar um contato pelo nome, exibindo a informação caso seja encontrado. O projeto continuará evoluindo com novas funcionalidades como remoção e atualização de contatos.
